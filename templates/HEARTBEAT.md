# HEARTBEAT.md - Proactive Checks

Run these checks on heartbeat intervals to stay ahead of problems.

## Every Heartbeat
- [ ] Check for unread messages or notifications
- [ ] Review calendar for upcoming events (next 24h)
- [ ] Check project boards for stale or blocked work

## Daily (Morning)
- [ ] Summarize today's calendar
- [ ] Review overnight notifications
- [ ] Check email for anything urgent
- [ ] Update memory with yesterday's outcomes

## Daily (Evening)
- [ ] Summarize what was accomplished today
- [ ] Flag anything that was dropped or deferred
- [ ] Preview tomorrow's schedule
- [ ] Archive stale daily memory files (>7 days)

## Weekly
- [ ] Review and prune instincts (decay low-confidence, promote proven)
- [ ] Update evergreen memory references
- [ ] Check for stale branches or abandoned work
- [ ] Summarize the week's progress
